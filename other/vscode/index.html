    VS Code 使用指北¶

    1.安装¶

    Link

    一般直接点击蓝色大方块即可

    然后运行安装程序安装

    2.切换中文¶

    一开始 VS Code 默认英文

    1. 打开侧边扩展按钮
    2. 搜索 Chinese
    3. 第一个（中文简体）安装
    4. 右下角会提示 Restart 重启应用即可获得中文页面

    具体见下图

    3.配置 C++ 环境¶

    推荐一个工具 VS Code Config Helper 4 (vscch.tk)

    下载后运行，跟着步骤即可

    完成之后打开 VS Code 会发现有个 helloworld.cpp 里面有注意事项，可以 F6 运行看看可不可以。

    注意事项¶

    1. 代码需要保存在你所选的文件夹中
    2. 文件名不可以有中文和特殊符号
    3. 更多内容可见 VS Code Config Helper v4 文档

    基础部分到此结束
